# ⚡ ClipBolt
**Seamless, Private Clipboard Sync between Windows & iPhone.**

ClipBolt uses a local FastAPI relay and Tailscale's secure WireGuard tunnel to sync your clipboard without your data ever leaving your private network.

## ✨ Features
- **Privacy-First:** No cloud servers. Data stays on your devices.
- **Ultra-Light:** Minimal CPU/RAM footprint.
- **Universal:** Works across any Wi-Fi network via Tailscale.

## 🚀 Quick Start
1. **Download:** Grab the latest `ClipBolt.exe` from the [Releases](#) page.
2. **Setup PC:** Run the `.exe`. It will create a `config.txt`. Enter your Tailscale name.
3. **Setup iOS:** Install the [iOS Shortcut](LINK_HERE).
4. **Sync:** Use the Shortcut to send/receive clips instantly.

## 🛠️ Built With
- Python (FastAPI + Pyperclip)
- Tailscale (Networking)
- Apple Shortcuts (iOS Integration)
